# Website Generation Engine

The deterministic website engine turns the structured business concept,
customer analysis, and strategy recommendations into an editable five-page
website package.

It produces:

- homepage, about, products/services, contact, and FAQ content
- local SEO and LocalBusiness JSON-LD only when location or service-area input exists
- Google Business Profile and social bio drafts
- a tone-specific brand-voice guide
- escaped static HTML with fixed CSS theme tokens
- a component-based Next.js export
- copy blocks for Squarespace, Wix, Shopify, and WordPress

Generated copy remains a founder-review draft. The engine neutralizes common
unsupported superiority and guarantee language, but every claim still requires
review before publication.
