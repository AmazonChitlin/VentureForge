# Feasibility Engine

`FeasibilityEngine.evaluate(projectInput)` is the deterministic third stage of
the VentureForge pipeline. It evaluates fifteen practical feasibility
categories, records the strongest proof-of-concept stage, and returns a
recommendation through the shared `EngineResult` envelope.

Optional evidence contracts are local adapters for future market research,
competitor analysis, financial, funding, and state-program modules. Missing
evidence reduces confidence. Scores are planning estimates, not guarantees,
professional advice, or replacements for official agency verification.
